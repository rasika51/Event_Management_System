To use this favicon:

  1. Take a screenshot of the "E" favicon above
  2. Use an online favicon generator to convert it to .ico format
  3. Replace the existing favicon.ico file
  4. Clear your browser cache (Ctrl+F5 or Ctrl+Shift+R)

Alternative: Use the SVG favicon

The SVG favicon should work in modern browsers. If you're still seeing the old favicon:

  • Hard refresh your browser (Ctrl+F5)
  • Clear browser cache
  • Try opening in an incognito/private window

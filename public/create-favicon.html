EventPro Favicon Generator

To create the favicon.ico file:

  1. Go to favicon.io or realfavicongenerator.net
  2. Upload the favicon.svg file from this directory
  3. Download the generated favicon.ico file
  4. Replace the existing favicon.ico in the public folder
